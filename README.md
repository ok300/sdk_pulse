# sdk_pulse

Create a `pulse-config.toml` file in this directory with

```toml
breez_api_key = "..."
sdk_1_mnemonic = "..."
sdk_2_mnemonic = "..."
iterations_csv_full_path = "..."
iterations_logs_dir_path = "..."
ln_address_wos = "..." # user@walletofsatoshi.com
ln_address_tor_node = "..." # user@domain.com
```

Then run with `cargo run`.
